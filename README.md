# Norstella Healthcare Data Portfolio

This repository contains 4 projects simulating **real-world healthcare data tasks**, aligned with the Norstella Data Quality Control Specialist role.  
All projects use **Python**, **pandas**, **NumPy**, and other libraries to handle **EMR, Claims, and Lab datasets** with realistic structures, anomalies, and validations.

---

## **Projects Overview**

### **Project 1: Healthcare Data Validation Dashboard**
- Validate EMR, Claims, and Lab datasets for duplicates, missing values, and anomalies
- Visualize patient ages, lab results, and claims distributions
- Skills: Python, pandas, NumPy, matplotlib, seaborn

### **Project 2: Data Schema Impact Analysis**
- Simulate schema changes (add/remove/rename columns) and analyze downstream impact
- Ensure robustness of healthcare data pipelines
- Skills: Python, pandas, data validation

### **Project 3: Automated Data Quality Checks Framework**
- Automatically detect missing values, duplicates, negative claim amounts, and anomalies
- Generate structured data quality reports
- Skills: Python, pandas, data quality automation

### **Project 4: Real-World Data Vendor Validation**
- Compare datasets from multiple vendors and detect mismatched claims
- Ensure data integrity before ingestion
- Skills: Python, pandas, data validation, vendor data comparison
