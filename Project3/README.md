# Project 3: Automated Data Quality Checks Framework

## Overview
This project implements automated data quality checks for healthcare claims data.  
It demonstrates how to generate a quality report that highlights issues before delivery.

## Key Features
- Detect missing values and duplicates
- Identify negative claim amounts and high-value outliers
- Generate a structured data quality report
- Automate repetitive validation tasks

## Skills Demonstrated
- Python (pandas, NumPy)
- Data Quality Automation
- Real-World Data Validation
- Reporting and Problem-Solving

## How to Run
1. Open `notebook.ipynb` in Google Colab or Jupyter Notebook
2. Run all cells to generate claims data and quality reports
